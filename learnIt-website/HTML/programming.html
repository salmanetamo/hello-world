Course Description

This course uses BlueJ, which is a simple and friendly integrated development environment, an IDE, to review some concepts that are core to the to Java programming language such as: data types, expressions, decisions, loops, and using the IDE debugging feature. It will then move on to discussing classes, objects, and data structures such as arrays and arraylists. So be ready to learn basic techniques such as managing loops and creating methods, debugging Java code, working with classes and objects, and storing and sorting arrays.


The following is the full list of all the topics that are explored in the Programming 1 module. They aim at introducing you to the basics of programming general with a particular focus on object oriented programming in one one of the most widely used programming languages that is Java :


  1. Intro to Object-Oriented Programming
  2. Writing Java code
  3. Defining classes
  4. Adding methods to classes
  5. Object Interaction
  6. Writing methods-more types, control flow and arrays
  7. Algorithms and method calls
  8. Development process and models
  9. Developing, Testing and Documenting Classes
  10. Library Classes, Refactoring and Inheritance
  11. Catalogue of code patterns
  12. Building Graphical User Interfaces
