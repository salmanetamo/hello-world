Introduction to Networking Technology

Instructors:
  • Maria Filippopoulou, Module leader
  • Stephen Naicken
Number of topics: 12

Course Description

This module provides an introduction to the basics of data networks operation and configuration. On completion of this module, students should be able to: Carry out basic configurations of common networking devices - Describe the typical operation of standard networks - Describe the journey and related processes of packets travelling over Ethernet and IP networks - Give reasonably detailed descriptions on the principle protocols underpinning network operation - Demonstrate understanding of the OSI and the TCP/IP stacks - Carry out IPv4 sub-netting exercises - Implement operation networks in the lab.
It is also hoped that the student will feel confident in future Network studies at university and beyond.

The following is alist of some of the topics that will be explored in the Introduction to Networks module. This will get you started with basic configurations of some networking devices, and also explores the different layers of the OSI model.

  1. Explore the network
  2. Configure a Network Operating System
  3. Network Protocols and Communications
  4. Network Access
  5. Ethernet
  6. Network Layer
  7. IP Addressing
